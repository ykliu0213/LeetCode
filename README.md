# LeetCode
### TwoSum
> * **Test.java** : testing program.  
> * **TwoSum.java** : brute-force solution.  
>   * Time complexity : O(n^2)  
>   * Space complexity : O(1)
> * **TwoSumHashMap.java** : better solution through HashMap.  
>   * Time complexity : O(n)  
>   * Space complexity : O(n)
